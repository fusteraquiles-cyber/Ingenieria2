# Ingeniería de Software II - Tecnicatura Superior en Desarrollo de Software

Este repositorio contiene los proyectos, trabajos prácticos y ejercicios desarrollados para la cátedra de **Ingeniería de Software II** (2do Año). El enfoque principal de este ciclo es la implementación de soluciones web modernas, aplicando metodologías de ingeniería y buenas prácticas de desarrollo.

---

## 🚀 Descripción del Proyecto

El objetivo de este espacio es documentar la evolución técnica en el desarrollo *frontend*, integrando maquetación avanzada, estilos responsivos y lógica de programación del lado del cliente. Se hace especial énfasis en la arquitectura de la información y la experiencia de usuario (UX).

## 🛠️ Tecnologías Utilizadas

El desarrollo se basa en el stack estándar de tecnologías web:

* **HTML5:** Estructura semántica de las aplicaciones.
* **CSS3:** Estilo y diseño visual personalizado.
* **JavaScript (ES6+):** Lógica de interacción, manipulación del DOM y consumo de datos.
* **Bootstrap 5:** Framework de diseño para garantizar la compatibilidad móvil y agilizar el desarrollo de interfaces.

## 📋 Contenido del Repositorio

A lo largo del cursado, se abordan los siguientes hitos:

1.  **Maquetación Semántica:** Estructuras limpias y accesibles.
2.  **Diseño Responsivo:** Uso de *Grid* y *Flexbox* (Bootstrap) para adaptabilidad en pantallas.
3.  **Interactividad con JS:** Validación de formularios, eventos de usuario y dinamismo.
4.  **Principios de Ingeniería:** Modularización del código y gestión de versiones con Git.
